FanciFrame examples!

Let's embed iframes with grace and beauty

Take the anchor and pass it as an anchor (it's the default!)

FanciFrame.embed("embedded.html");

Take the anchor and pass it as an anchor

FanciFrame.embed("embedded.html", { input: 'anchor', output: 'anchor' } );

Pass parameters to the iframe

FanciFrame.embed("embedded.html", { input: 'anchor', output: 'anchor' }, { 'class': 'fun-iframe', style: 'width: 600px' } );

Take the params and pass them as an anchor

FanciFrame.embed("embedded.html", { input: 'params', output: 'anchor' } );

Take the params and pass them as params

FanciFrame.embed("embedded.html", { input: 'params', output: 'params' } );

Take the params and pass them as params (appending to existing params)

FanciFrame.embed("embedded.html?test=true", { input: 'params', output: 'params' } );

Take the value of a specific parameter (state) and pass it as a specific parameter (selected)

FanciFrame.embed("embedded.html", { input: 'state', output: 'selected' } );

Take the value of a specific parameter and pass it as an anchor

FanciFrame.embed("embedded.html", { input: 'state', output: 'anchor' } );

Take the value of a specific parameter and pass it as the same name

FanciFrame.embed("embedded.html", { input: 'state', output: 'params' } );

Take the value of a specific parameter and pass it as a hash

FanciFrame.embed("embedded.html", { input: 'state', output: 'anchor' } );
